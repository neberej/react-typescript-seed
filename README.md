# A no-nonsense seed project using React, Express and Typescript


Features:

- Typescript
- Webpack 5
- React 18
- Express 4 (Optional)
- Tailwind

Sub-Features:

- Fully configured for testing using Jest
- Eslint setup

---

### Run front-end

- Development

```
npm run dev
```

- Build

```
npm run build
```

### Run Back-end

```
npm run server
```


### Run Test

```
npm run test
```


### Run Eslint

```
npm run lint
```
